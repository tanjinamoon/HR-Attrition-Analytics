## Project Overview
This project is an end-to-end HR analytics capstone focused on understanding employee attrition and workforce trends. The analysis combines data exploration, modeling, and an interactive Streamlit application to deliver actionable insights for workforce planning and retention strategies.

---

## Business Problem
Organizations struggle to identify why employees leave and how attrition varies across roles, locations, and workforce segments. The objective of this project was to:
- Identify key drivers of employee attrition
- Compare attrition patterns across employee groups
- Provide an interactive tool for decision-makers to explore insights

---

## Approach
- Data cleaning and feature engineering
- Exploratory analysis of workforce and attrition patterns
- Segmentation of employees based on key characteristics
- Development of an interactive Streamlit app for insight exploration

---

## Key Insights
- Attrition patterns vary significantly across employee types and locations
- Certain workforce segments show consistently higher attrition risk
- Data-driven segmentation supports more targeted retention strategies
- Interactive dashboards improve accessibility for non-technical stakeholders
